Project Case Study: SocioScore

The Problem

Traditional reputation tracking in organizations is often manual and biased. There was no automated way to measure reliability based on actual data and task completion.

The Solution

SocioScore automates this using a custom algorithm. It tracks user activities and calculates a 'Trust Score' that promotes accountability.

Tech Stack

  • Backend: Java 21, Spring Boot
  • Security: Spring Security (Auth)
  • Database: MySQL
  • Deployment: Docker Containers

Key Features

Real-time Scoring Admin Dashboard REST API Integration Dockerized Setup

Project Case Study: XplorIndia

The Concept

XplorIndia was developed to bridge the gap between new students/tourists and city resources. It serves as a one-stop digital directory for navigating city-specific amenities.

Core Objective

To provide a user-friendly interface where users can find verified information about local institutes, hospitals, and tourist attractions without multiple web searches.

Tech Stack

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Java (Servlets & JSP)
  • Database: MySQL
  • Tools: Maven, Apache Tomcat

Key Features

Category-based Filtering Location Mapping User Feedback System Responsive Directory
